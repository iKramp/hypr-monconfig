# Hyprland monitor configuration tool

This tool is very much a work in progress, but the target is a fully feature complete GUI tool that manages monitors.

## planned features
- [ ] Moitor placement and rotation
- [ ] Resultion, fps and scale setting per monitor
- [ ] Workspace-monitor setting (eg. binding workspaces from 1 to 3 to monitor DP-1 and the rest to DP-2...)
- [ ] Live updating (on clicking apply) with automatic reverts if the user can't confirm the config is ok
- [ ] Separate monitor configuration file in XDG_CONFIG_HOME/hypr/hypr-monconfig.conf that can be included in the hyprland config
- [ ] Scanning of the main + linked hyprland configs for existing monitor configuraion that should be done with the tool instead
